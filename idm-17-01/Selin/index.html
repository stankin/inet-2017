Подготовка к экзамену

Формулировка одного вопроса по теме "Интернет-технологии" в одной из форм, встречающихся в тесте (выбор одного, выбор многих, упорядочивание, соответствие)

Вопрос: Что задает команда ip route 186.157.5.0 255.255.255.0 10.1.1.3
Правильный ответ: Маршрутизатор должен использовать 10.1.1.3, чтобы получить доступ к сети 186.157.5.0 Неверный вариант ответа: Маршрутизатор должен использовать сеть 186.157.5.0,чтобы получить доступ к 10.1.1.3 Неверный вариант ответа: Для 186.157.5.0 и 10.1.1.3 используется маска 255.255.255.0 Неверный вариант ответа: Маршрутизатор должен трассировать маршрут к сети 186.157.5.0 через 10.1.1.3
