Лабораторная работа #3. Настройка локальной сети передачи данных

  1. Создание план IP-адресации небольшой сети. Проектируемая сеть состоит из 3 LAN'ов с заданным количеством узлов. Для организации связи между устройствами в одной локальной сети используются коммутаторы, для организации связи между локальными сетями - маршрутизаторы.
